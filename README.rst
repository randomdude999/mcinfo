======
mcinfo
======

A command-line tool to show information about Minecraft blocks, items and more.

Usage
-----

::
    $ mcinfo stone
    [TBI]
    $ mcinfo iron_sword
    [TBI]
    $ mcinfo pig
    [TBI]
    $ mcinfo
    > nbt:cow
    [TBI]
    > wooden_sword
    [TBI]
    > quit

That's kind of it.
